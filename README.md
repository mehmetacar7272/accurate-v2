# ACCURATE V2 Sprint 1 Starter

## Backend
```powershell
cd backend
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Frontend
```powershell
cd frontend
npm install
npm run dev
```
